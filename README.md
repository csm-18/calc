# calc

calc is a cli calculator written in rust.

It supports:

    +   -> addition

    -   -> subtraction

    *   -> multiplication

    /   -> division

    %   -> remainder

    ()  -> bracket expressions

## Setup

1. clone the repo
2. cd into calc directory
3. run:
   `cargo run`
